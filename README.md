# NLCS_ATTT
#Đây là đề tài sử dụng C# (C shape) để thiết kế bao gồm 2 phần
1. Một chương trình keylogger gồm các chức năng: chụp màn hình, ghi nhận phím và chuột, lấy registry, lấy tiến trình hệ thống, gửi báo cáo qua mail,...
2. Một giao diện để import vào file zip từ đó hiển thị các chức năng được ghi nhận ở keylogger và thể hiện một cách trực quan hóa.
